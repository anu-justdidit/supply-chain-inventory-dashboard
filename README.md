
# Supply Chain Inventory Dashboard

## Business Problem
Frequent stockouts were causing lost revenue and customer dissatisfaction. The goal was to reduce stockouts and improve inventory efficiency.

## Solution
A real-time Power BI dashboard was developed to track inventory levels, product movement, and key metrics.

## Key Metric
**Inventory Turnover** – measures how efficiently inventory is sold.

## Methodology
- **Data Extraction**: SQL queries from transactional databases.
- **Data Modeling**: Star schema in Power BI.
- **DAX Measures**: Stock levels, turnover ratio, alerts.
- **Real-time Tracking**: Direct connection to data source for up‑to‑the‑minute visibility.

## Results & Business Impact
Enabled proactive inventory management, leading to a **15% reduction in stockouts**.




I built a Power BI dashboard that tracks inventory turnover in real time. The measure Inventory Turnover = Units Sold / Average Inventory updates instantly. When turnover fell below a threshold, products were flagged as ‘slow movers’. The supply chain team used this to cut orders and run promotions, which freed up storage and reduced stockouts from 12% to 10.2% – a 15% reduction.
